阿里巴巴B2B（香港上市部分）：雅虎持有约30%股份，价值每股 2.1美元。

合计： 4.6 + 2.1 = 6.7美元 。

公式： 每股投资资产价值 = ∑(所持上市公司股权市值 × 持股比例) / 总股本 。

3. 计算每股核心业务价值

概念：评估雅虎自身主营业务（美国雅虎）的盈利能力。这里段永平采用了市盈率（PE）法。

段永平的计算：

确定每股盈利（EPS）：他估算雅虎核心业务每股盈利约为 0.8美元。

选择市盈率倍数：他给这部分业务赋予了 12倍PE。这个倍数是基于对互联网广告市场成长性的判断，可以看作是“看涨期权”。

计算价值： 核心业务价值 = EPS × PE = 0.8美元 × 12 = 9.6美元 。

公式： 每股核心业务价值 = 每股盈利 × 合理的PE倍数 。

4. 计算基础价值总和

将以上三部分相加，得到雅虎的“基础”内在价值：

 每股基础价值 = 每股净现金 + 每股投资资产价值 + 每股核心业务价值 = 3 + 6.7 + 9.6 = 19.3美元 

因此，段永平认为雅虎的公平价值（fair value）大约在18-20美元之间。在这个价格下，雅虎已经“不贵”。

5. 考虑隐性资产（阿里巴巴未上市部分）的价值

这是段永平投资雅虎的“最大理由”，他将这部分视为免费看涨期权或安全边际。

概念：雅虎当时持有阿里巴巴集团（包括淘宝、支付宝等未上市业务）40%的股份。这部分资产由于未上市，市场难以准确定价，因此被很多投资者忽略。

段永平的场景分析：

场景一：如果整个阿里巴巴集团未上市部分价值 500亿美元，那么雅虎持有的40%股权就价值200亿美元。将这200亿美元的价值分摊到总股本中，会使雅虎的股价超过30美元。

场景二：如果阿里巴巴集团只值 200亿美元，雅虎的持股价值80亿美元，分摊后股价大约在25-26美元。

逻辑：由于他在19.3美元的基础价值中并未计入这部分资产，所以无论阿里巴巴最终值多少钱，对雅虎股东来说都是“白送”的额外价值。这正是投资的安全边际和潜在利润来源。
