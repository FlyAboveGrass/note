### 乘法

1. 小分互换。小数和分数互换，判断大致的范围
2. 乘法拆分。拆成简单的数的相加或相减，拆分律
	1. 678 * 45% = 678 * (50% - 5%)
3. 乘法放缩。乘法拆分的拓展
	1. 462 * 204 = 462 * (200 + 4) = 462 * 200(1 + 2%)

### 除法

1. 拆分法。补到普通的分数附近，例如

462 / 986 = （493 - 31） / 986 = 1/2 - 31/986

2. 除法缩放。上下都加或者减一个等比例的数，目的是让分母变成整十、整百、整千
 365 / 986 ≈ （356 + 5） / （986 + 14） ≈ 361 / 10000
850 / 1031 ≈ （850 - 25）/（1031 - 31） ≈ 825 / 1000
3. 截位直除法。在选项差距比较大时，直接使用近似数来计算。例如： 6.8÷(1+4.5%)≈6.8÷1.05

## **ABRX 四量关系**

- **A (基期量)​**​：作为对比**基准**的时期的数值。例如，在表述“2024年比2023年增长……”时，​**2023年的数据就是基期量**
- ​**B (现期量)​**​：当前**所研究时期**的数值。在上面的例子中，​**2024年的数据就是现期量**
- ​**R (增长率)​**​：表示现期量相对于基期量的**相对变化幅度**，通常用百分数表示。计算公式为 `R = (B - A) / A`，它也等价于 `(B/A) - 1`
- ​**X (增长量)​**​：表示现期量相对于基期量的**绝对变化数量**。计算公式为 `X = B - A`

`R = (B - A) / A = X / A`

### 速算技巧 —— 415 份数法

把数量关系转化成比例关系（分数），简化分数。
415 份数法，"415" 分别代表基期、变化量和现期。

假设增长率为 25%，由 4 可以快速得到： 变化量 X = 4 * 1/4  = 1；现期 = 4 + 1 = 5.
反过来同理： 当现期为 5 ，增长率为 25%。可知 B 是 A 的 5/4 倍，那么假设将现期 B 拆成 5 份，则 A 占 4 份，增长率占 1 份。

例题： 现期 B = 954，增长率 R = -14.4%，求基期 A 和增长量 X。

先求增长量：
增长率 R ≈ -1/7，意思就是 B = 6/7 * A，X = 1/7 A。意思就是 X / B = 1/6， X = 954 / 6 = 159

基期 A  = B + X = 954 + 159 = 1113

#### 在计算过程中，纠正误差

> 基础原则是，在偏差率的基础上乘以 （份数 - 2）。例如分成 7 份，偏差率为 0.9%，则纠偏的比例应该为 （7-2）x 0.9% = 4.5.

R 差修正：
1. 5 ～9 ： 都是 x - 2
2. 4 ： 2.2
3. 3 ： 1.3
4. 2 : 0.5


例题： B = 642， R = 26%。

<u>将 R 粗略认为是 1/4，则将 B 拆成 5 份，X 占 1 份，A 占 4 份。</u>


X = 642 / 5 = 128

由于 26% 和 1/4 存在 1% 的偏差，需要进行纠偏。按基础原则，偏差率为 1% + （5 -2） = 3%

纠偏量 = 128 x 3% ≈ 3.84
实际计算 X = 128 + 3.84 ≈ 132
A = 642 - 132 = 510

## 比重类

### 概念

部分增长率： Ra
总体增长率：Rb
基期比重 A = 基期部分 / 基期总量
现期比重 B = 部分 / 整体 = A x (1 + Ra) / (1 + Rb)
- 反之 基期比重 A： 现期比重 B x (1 + Rb) / (1 + Ra)
- 推导可得 比重趋势 X：
	- Ra > Rb 比重上升；Ra < Rb 比重下降
	- 反推： 比重上升，Ra > Rb; 比重下降， Ra < Rb
比重的差（现期比重 B - 基期比重 A） = m (Ra - Rb) / n(1 + Rb)= 基期比重 A * (Ra - Rb) / (1 + Rb) = 基期部分m / 现期总量 *  (Ra - Rb)
- <u>比重的增长率： (Ra - Rb) / (1 + Rb)</u>
- 大部分情况下，基期部分都小于现期总量，可得： 增速差 > 比重差. 即 Ra - Rb > B - A

#### 经典题型

比重的对比：

去年比重  m/n, 今年变化后 m (1 + Ra) / n (1 + Rb)

- 两期比重比较
- 比重趋势变化。直接看 Ra 和 Rb 的对比，就可以看出趋势， Ra >  Rb 则 (1 + Ra) /  (1 + rb)  小于 1，就是下降，反之上升。
	- 分子增速大于分母，则分数变大，比重上升
	- 分子增速小于分母，则分数变小，比重下降
- 根据比重变化和现期比重，推测过去比重
- 比重的增长率： (Ra - Rb) / (1 + Rb)

## 盐水思想（混合思维）

### 概念

- 总量 A， A 的平均值 R1
- 另一总量 B， B 的平均值 R2
- 混合后的总量 A + B，混合后的平均值 r = (A x R1 + B x R2) / (A + B)
	- <u>A / B = (R2 - R) / (R - R1)</u>
- 混合之后，谁的总量大，新的平均值就靠近谁

### 计算法则

混合后的平均值 R = (A x R1 + B x R2) / (A + B)
	<u>A / B = (R2 - R) / (R - R1)</u>




- 十字交叉法
- 线段法
	- 根据份数拆分去算 R 会更简单， <u>A / B = (R2 - R) / (R - R1)</u>
	- 方法
		1. 划线标点
		2. 量之反比
		3. 几何计算

### 混合类题目的题型

- 三排序问题。 R1 - R - R2
	- 从其中两个的比重，求另外一个比重
- 已知 3 个比重，求量之间的比
- 已知 2 个比重和量之比，求第三个比重
	- 线段法。先定性，再定量。平均比重一定是靠近量大的一边

## 论证类

### 归因论证

观察者对于行为过程所进行的因果解释和推论。

**分类**
1. 实验对比归因
	1. 三圈质疑 and 四圈支持
	2. **例子**​：一项研究将受试者分为两组，一组每天食用大量白米饭，另一组食用少量白米饭。结果发现，食用量大的人群患糖尿病风险显著更高，研究者据此得出结论：多吃白米饭易患糖尿病
2. 时间对比归因
	1. **例子**​：某位同学最近成绩突飞猛进，大家认为是因为他参加了某个培训班（茉莉培训班）。这就是一个时间对比归因——将“成绩提高”（当前变化）归因于“参加培训班”（近期变化
3. 一般归因
	1. 有研究者认为“儿童缺钙会导致脾气暴躁”。（这里，“脾气暴躁”是既成事实，研究者直接将其归因于“缺钙”，没有涉及与其他儿童（实验对比）或不同时期（时间对比）的明确比较。）
4. 直接根本原因
	1. 陈胜吴广起义，其直接原因是遇大雨延误行程面临处罚，而根本原因是秦朝的严苛制度。
5. 题干和选项构成对比实验。



质疑方式，首先确定因果，然后一步步找方式质疑：
1. 另有他因。在题干认为的原因之外，还有其他可能导致结果的原因。
	1. <u>回归实验。</u>题干是通过实验得到的结论，如果要质疑它，那就回归到实验中，找实验的漏洞。
	2. 继发关联。 A 和 B 共同出现，不代表 A 和 B 存在因果关系，而是 A 和 B 都有共同的原因 C。
2. 因果倒置。  A 和 B 共同出现，不代表 A 是因 B 是果，也有可能 B 是因 A 是果。
3. 否定此因。不谈原因是什么，但是不是这个原因。

支持方式：
1. 排除他因
2. 解释说明

### 一般质疑

“一般质疑”主要针对的是**非归因类**的论证。它核心关注的是论证过程的合理性，而非具体的因果关系。<u>你需要做的就是**找出论证中的漏洞**，并选择最能攻击这个漏洞的选项。</u>

**答题步骤建议**​：

- ​**读提问**​：明确是“最能质疑/削弱”的。
- ​**找结论和论据**​：准确识别出题干的结论是什么，支持结论的论据又是什么。
- ​**分析论证过程**​：思考论据是如何推导出结论的，这个推导过程有什么内在缺陷或隐含假设。
- ​**评估选项**​：将每个选项带入“虽然论据成立，但是…”的句式，看是否能合理得出“结论不成立”的结论。优先选择**针对性最强、力度最强**的选项。

#### 题型和解题思路

- **质疑论证过程**（占比 90%）​
	- **肯定论据，但指出论据推不出结论**​（常用<u>“虽然…但是…”</u>逻辑。
		- 题目 A 推出 B，就去找 A 推非 B 来反驳。（例如： 努力就会上岸，反驳就可以说： 虽然某某努力了，但是也没上岸）
	- **指出范围不一致**​。​**指出论据与结论的范围、主体或概念不匹配**。以偏概全/存在断点
	- **考虑不全面**​。指出论证忽略了的其他重要因素或负面后果
- **质疑论据本身**​
	- **直接指出支持结论的证据不成立或不可靠**
	- **提供反面证据**​。提出与结论直接相反或矛盾的事实
	- **另有他因**​。​**指出存在其他更主要或同时存在的原因导致结果**
- 质疑结论。
	- 结论中有无中生有的内容，夹带私货

#### **💡 实用技巧与常见陷阱**​

- ​**关键词捕捉**​：注意结论前的引导词，如“因此”、“所以”、“据此认为”等。留意选项中的程度词，如“有些”、“部分”通常**削弱力度较弱**；“主要”、“根本”、“无法”等词则可能指向更强力的质疑
- ​**比较选项力度**​：通常，“直接否定结论或论证关系”比“间接质疑”力度强；“整体性质疑”比“个别案例质疑”（如用“有些”）力度强；“针对性质疑”比“泛泛而谈”力度强。
- ​**警惕常见错误选项**​：
    - ​**无关项**​：选项讨论的内容与题干结论无关或话题不一致
    - ​**诉诸未来或不确定性**​：如“还需要进一步研究
    - ​**主观意见或价值判断**​：如“这种方法不道德”
    - ​**加强项**​：注意提问是“削弱”还是“加强”，别选成支持结论的。

### 比例类论证

比例类论证： 此类问题常犯的错误是用 "分子的多少" 来代替 "分子/分母的比例" 得出结论，题干往往包含数字（分子部分），我们在选项中找到带数字的"分母"部分，识别题型，就可以秒杀。

### 支持论证

常见支持方式：
- 解释说明
- 断点搭桥
	- 题干结论存在论据中没有的新内容，此新内容一定要在选项中有所体现，并能够和论据中的关键信息进行搭桥
- 增加正向论据
- 必要条件
	- 力度不是很强的选项。是结论成立的必要前提，没有这个前提推理一定是错误的。可以用反向代入的方式去验证选项，情况与这个选项相反，则得不出题干的结论，那么这个选项就是结论的必要条件。
	- 例如： 某起盗窃案的主犯是 A。其必要的前提是，A 当天在该地出现。
- 举例支持

#### 支持论证类

常见问法： "以下哪一项如果为真，最能加强上述论证"

**tips：**
- 要着重变化！有变化才能显出论证对比和正确性。
- 比较要说两边。两个维度都要比较，不能仅从单一角度论证。

#### 前提假设类

常见问法： “如果上述结论为真，需要补充的前提是”；"上述推测海需要隐含哪一项假设"


三种解题方案：
- 断点搭桥。
	- 论据 ABC，结论 F。那么我们需要进行断点搭桥，找到能把结论和论据能搭边的选项 CF。
- 补充漏洞。
	- 逻辑上存在漏洞，需要补足。
- 能与不能。
	- 没有这个条件，推理无法进行或者结论无法成立。

#### 解释说明类

常见问法： “以下最能解释这一现象的是”


**重点： 用变化解释变化!!!**
