# 图形推理总结

## 1. 宏观原则

- ​**规则优先看位置**​：在分析图形时，首先观察图形的位置变化。
    
- ​**对称性**​：对称性在图形推理中出现频率很高（约95%），应作为首要考虑因素。
    
- ​**修正思路**​：当直接规律走不通时，需要修正思路，考虑其他角度。
    

## 2. 图形元素分析

- ​**基本元素**​：
    
    - 图形分离时，考虑元素数量或种类。
        
    - 纯封闭图形优先考虑面、出头点或点。
        
    
- ​**常见考点**​：
    
    - 角的变化（如角的数量或类型）。
        
    - 曲线和圆形（圆形常作为提示，考虑圆内图形、切点、交点等）。
        
    - 生活类图形（如字母、标志）考虑笔画、开闭曲直等。
        
    - 同心圆或单个图形可能考察对称性或元素传递。
        
    

## 3. 具体考点列表

- ​**面相关**​：
    
    - 阴影出现时，常考面（而非笔画）。
        
    - 最大面的属性（对称性、形状、边数）。
        
    - 相同面或面的遍历（所有图形中面的一致性）。
        
    - 面之间的公共边或连接方式（如三面重合、公共边条数）。
        
    
- ​**线相关**​：
    
    - 纯直线图形考虑面、线、角、交点的顺序。
        
    - 平行线（本质是考线）或内外框的角的数量。
        
    - 线的连接方式（点连接、线连接，如电线杆模型）。
        
    
- ​**点相关**​：
    
    - 交点数变化（如圆上交点、圆内交点）。
        
    - 点连接与线结合。
        
    
- ​**黑白运算**​：
    
    - 黑白方块或黑白球考虑运算规则（如叠加、替换）。
        
    - 九宫格中常见三推三规律，阴影数不一致时考虑平移或传递。
        
    - 注意白色部分易被忽略，尤其有两种阴影时。
        
    
- ​**位置关系**​：
    
    - 图形覆盖（上下左右、相交面属性）。
        
    - 空间位置关系（如公共边、公共线）。
        
    - 对角线上的图形连接或相似性。
        
    
- ​**元素传递**​：
    
    - 部分数或相同元素数量。
        
    - 线条状图形（如T、Y型）考虑重复数或分割。
        
    - 火柴棍模型数数或位置变化。
        
    

## 4. 解题步骤和优先级

1. ​**先观察位置**​：查看图形排列顺序（如行、列、对角线）。
    
2. ​**考虑对称性**​：检查轴对称、中心对称等。
    
3. ​**分析元素**​：从面、线、角、点入手，顺序为面→线→角→交点。
    
4. ​**特殊图形处理**​：
    
    - 有圆时，考虑圆上点、圆内点、切点、交点。
        
    - 内外框图形，分析外线内面或外线内线关系。
        
    
5. ​**运算规则**​：如果图形有叠加，尝试图1+图3=图2或隔图看规律。
    
6. ​**修正思路**​：如果常规规律无效，考虑修正（如交点数、笔画数）。
    

## 5. 注意事项

- ​**常见陷阱**​：
    
    - 阴影图形不考笔画，常考面。
        
    - 最大面形状相似时，考对称性位置。
        
    - 黑白运算中，中间位置黑白交替常见。
        
    
- ​**图形特征提示**​：
    
    - 图形画得规整时，考虑对称性或位置。
        
    - 多个圆带线图形，优先考虑圆上交点数、圆内线或面、笔画数。
    - 
- 形状
	- 大小
	- 对称
	- 几边形
	- 相似
- 位置
	- 与外轮廓相对
	- 与白块相对
- 对接
	- 几条边
	- 对接边的长短
	- 曲直
	- 方向
	- 对接方式（嵌入、锯齿）
